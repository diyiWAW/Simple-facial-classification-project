
-------------------------------npy文件---------------------------------------
pca_img.npy :使用主成分分析对人脸图像进行特征降维
data_projected.npy : 使用isomap对人脸图像进行特征降维
img.npy: 储存人脸图像的numpy数组，数组shape为(3993,16384)，
	即代表3993张图片，每张16384(128*128)个像素点
-----------------------------------------------------------------------------

---------------------------python文件-------------------------------------------
util.py:  读取标签文件、图片文件的函数
FaceDataset.py: 用pytorch制作神经网络数据集
ealyStop.py: 早停
rec.py : 使用knn bayes 决策树 svm和集成学习方法进行人脸识别
bp.py : 使用resnet18神经网络模型进行人脸识别
-------------------------------------------------------------------------------


--------------------------------其他文件--------------------------------------
label.txt : 人脸图像的标签，包含图像的sex, age, race, face, prop五个标签；
	id表示图像的文件名；missing表示该图像的标签是否存在缺失

best_network.pth : 由bp.py训练出来的性别分类神经网络模型

总结.docx : 总结
